# 🌌 Star Wars Characters App

Aplicación web creada con **React + Vite** que muestra personajes icónicos de Star Wars.

Incluye:

- 🔍 Buscador de personajes
- 🧑‍🚀 Tarjetas interactivas
- 📜 Modal con información del personaje
- 🎬 Animaciones al hacer scroll
- ⚡ Filtrado dinámico

---

# 🚀 Tecnologías usadas

- React
- Vite
- JavaScript
- CSS
- Intersection Observer
- Custom Hooks
