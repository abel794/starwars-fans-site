# 🌌 Star Wars Characters App

Aplicación web creada con **React + Vite** que muestra personajes icónicos de Star Wars.

Incluye:

- 🔍 Buscador de personajes
- 🧑‍🚀 Tarjetas interactivas
- 📜 Modal con información del personaje
- 🎬 Animaciones al hacer scroll
- ⚡ Filtrado dinámico

---

# 🚀 Tecnologías usadas

- React
- Vite
- JavaScript
- CSS
- Intersection Observer
- Custom Hooks

# ⚙️ Instalación

1️⃣ Clonar el repositorio

git clone https://github.com/abel794/starwars-fans-site.git

2️⃣ Entrar al proyecto

cd starwars-fans-site

3️⃣ Instalar dependencias

npm install

4️⃣ Ejecutar el proyecto

npm run dev
